# CZRPG
이 문서는 CZRPG에 대한 설명입니다.

## COMAND
### party
파티 생성
```
/party <create> <파티 이름>
```
플레이어를 파티원으로 초대
```
/party <invite> <플레이어 이름>
```
파티 삭제
```
/party <delete> <파티 이름>
```
## Function
### reinforce
인챈트테이블을 우클릭하여 강화GUI를 띄웁니다.

3번 slot에 강화할 아이템을 넣고 5번 slot에 강화석을 넣습니다.
//사진

강화 확률에 따라 강화 하거나 강화 실패가 될수 있습니다.

실패 시 일정 확률로 파괴

강화 확률
```
(20/강화 정도)%
```
실패확률
```
(-20/((강화 정도/2)-11)-0.4)%
```

### Join
서버에 입장할때 주문서를 줘 우클릭하여 사용할 수 있습니다.
